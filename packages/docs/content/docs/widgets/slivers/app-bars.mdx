Scrolling app bars that collapse, pin, float, and stretch. The Material `SliverAppBar` (with `FlexibleSpaceBar` for its expanded area) and the iOS `CupertinoSliverNavigationBar` both live as the first child of a [`CustomScrollView`](/docs/widgets/slivers).

## SliverAppBar

A Material app bar that scrolls, collapses, pins, floats, and stretches. Set `expandedHeight` and a `flexibleSpace` for the classic large-photo-collapses-to-toolbar effect. Maps to Flutter's [`SliverAppBar`](https://api.flutter.dev/flutter/material/SliverAppBar-class.html).

```tsx
import { FlexibleSpaceBar, Icon, SliverAppBar, Text, View } from "solid-fuse";
import { share, favorite } from "solid-fuse/icons/material";

<SliverAppBar
  pinned
  expandedHeight={240}
  title={<Text color="white">Mount Rainier</Text>}
  actions={[
    <Icon data={share} color="white" />,
    <Icon data={favorite} color="white" />,
  ]}
  flexibleSpace={
    <FlexibleSpaceBar
      background={
        <View
          width={Infinity}
          height={Infinity}
          decoration={{ image: { url: "rainier.jpg", fit: "cover" } }}
        />
      }
    />
  }
/>
```

Collapse behavior is fully declarative — `pinned`, `floating`, `snap`, and `stretch` describe what the bar does as you scroll; there's no shrink-offset callback to wire up. Use `type` for the Material 3 sizes (`"medium"` and `"large"` build the taller variants).

### actions is an array of nodes

`actions` takes a JSX **array**, not children — `actions={[<A />, <B />]}`. This is the array-of-nodes prop pattern: each element is an independent JSX node serialized into the prop, laid out in the bar's trailing slot. The same shape applies anywhere a slot holds a list of widgets rather than a single child.

| Slot | Type | Description |
|---|---|---|
| `leading` | `JSX` | Widget before the title (back button, menu) |
| `title` | `JSX` | Title widget |
| `flexibleSpace` | `JSX` | Expanded-area content, usually a `FlexibleSpaceBar` |
| `bottom` | `JSX` | Persistent area below the toolbar (tabs, a search field) |
| `actions` | `JSX[]` | Trailing action widgets — an array of nodes |

| Prop | Type | Description |
|---|---|---|
| `type` | `string` | `"small"` (default), `"medium"`, `"large"` |
| `pinned` | `boolean` | Stays at the top when fully collapsed |
| `floating` | `boolean` | Reappears on any upward scroll |
| `snap` | `boolean` | With `floating`, snaps fully open/closed (no partial state) |
| `stretch` | `boolean` | Over-scrolls past `expandedHeight` |
| `forceElevated` | `boolean` | Always show the elevation shadow |
| `primary` / `centerTitle` / `automaticallyImplyLeading` / `excludeHeaderSemantics` | `boolean` | Standard app-bar flags |
| `expandedHeight` | `number` | Height when fully expanded |
| `collapsedHeight` | `number` | Height when collapsed (above the toolbar) |
| `toolbarHeight` | `number` | Toolbar row height |
| `elevation` / `scrolledUnderElevation` | `number` | Shadow depth, resting and scrolled-under |
| `titleSpacing` / `leadingWidth` / `bottomOpacity` | `number` | Layout tuning |
| `stretchTriggerOffset` | `number` | Over-scroll distance that fires `onStretchTrigger` |
| `backgroundColor` / `foregroundColor` / `shadowColor` / `surfaceTintColor` | `ColorInput` | Colors |
| `systemOverlayStyle` | `string` | `"light"` or `"dark"` status-bar icons |
| `clipBehavior` | `string` | Clip behavior |
| `onStretchTrigger` | `() => Promise<void>` | Awaitable callback fired when stretched past the trigger |

## FlexibleSpaceBar

Goes in `SliverAppBar`'s `flexibleSpace`. Holds a `title` and a `background` that animate as the bar collapses. Maps to Flutter's [`FlexibleSpaceBar`](https://api.flutter.dev/flutter/material/FlexibleSpaceBar-class.html).

```tsx
import { FlexibleSpaceBar, Text, View } from "solid-fuse";

<FlexibleSpaceBar
  title={<Text color="white">Gallery</Text>}
  centerTitle
  collapseMode="parallax"
  stretchModes={["zoomBackground", "fadeTitle"]}
  background={
    <View
      width={Infinity}
      height={Infinity}
      decoration={{ image: { url: "cover.jpg", fit: "cover" } }}
    />
  }
/>
```

| Prop | Type | Description |
|---|---|---|
| `title` | `JSX` | Title widget, scales between expanded and collapsed |
| `background` | `JSX` | Background widget behind the title |
| `centerTitle` | `boolean` | Center the collapsed title |
| `titlePadding` | `EdgeInsetsInput` | Padding around the title |
| `expandedTitleScale` | `number` | Title scale factor when expanded |
| `collapseMode` | `string` | `"none"`, `"pin"`, `"parallax"` — how the background moves |
| `stretchModes` | `string[]` | Any of `"zoomBackground"`, `"blurBackground"`, `"fadeTitle"` |

## CupertinoSliverNavigationBar

`CupertinoSliverNavigationBar` is the iOS large-title bar that collapses into a standard nav bar as you scroll — the `SliverAppBar` of the Cupertino world. Maps to Flutter's [`CupertinoSliverNavigationBar`](https://api.flutter.dev/flutter/cupertino/CupertinoSliverNavigationBar-class.html).

```tsx
import { CupertinoSliverNavigationBar, CustomScrollView, SliverList, Text } from "solid-fuse";

<CustomScrollView>
  <CupertinoSliverNavigationBar largeTitle={<Text>Settings</Text>} />
  <SliverList>{/* rows */}</SliverList>
</CustomScrollView>
```

| Slot | Type | Description |
|---|---|---|
| `largeTitle` | `JSX` | Large title shown when expanded |
| `leading` | `JSX` | Leading widget |
| `middle` | `JSX` | Collapsed-state title |
| `trailing` | `JSX` | Trailing widget |
| `bottom` | `JSX` | Persistent area below the bar |

| Prop | Type | Description |
|---|---|---|
| `automaticallyImplyLeading` / `automaticallyImplyTitle` | `boolean` | Infer leading/title from context |
| `alwaysShowMiddle` | `boolean` | Keep the collapsed title visible while expanded |
| `stretch` | `boolean` | Over-scroll stretch |
| `transitionBetweenRoutes` | `boolean` | Hero-style transition between routes |
| `brightness` | `string` | `"light"` or `"dark"` |
| `backgroundColor` | `ColorInput` | Bar color |
| `border` | `BorderInput` | Bottom border |
| `padding` | `EdgeInsetsInput` | Content padding |
