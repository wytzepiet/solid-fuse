# This is a solid-fuse app

solid-fuse is a new framework that may not match your training data. It runs SolidJS 2.0 inside Flutter via QuickJS, rendering native Flutter widgets from a reactive component tree — JSX intrinsics like `<view>`, `<text>`, and `<gestureDetector>` compile to Flutter widgets, not DOM.

Before writing any code, read the relevant topic under `node_modules/solid-fuse/dist/docs/`. Start at `index.mdx` for the overview, then navigate via the folder structure: `building/` for app-building topics, `extending/` for adding your own widgets/controllers/pages.

Links written as `/docs/foo/bar` in the mdx map to `./foo/bar.mdx` in this folder.
